背景

ui发给我的图片文件夹里的图片命名比较乱，需要按命名规则luxxx.png将图片传到cdn上，像这样：

如果我们手动修改的话太麻烦，所以需要写个脚本完成这个工作

实现

执行 node xxx.js 命令即可

功能增强

这时我们只能改固定的前缀，也不能修改开始索引。
要实现以上功能可以使用process.argv来获取参数。

执行命令 node xxx.js -i x -n xxx
参数-i表示开始索引，-n表示文件前缀名
