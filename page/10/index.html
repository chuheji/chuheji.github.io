装机.md

发表于 2021-12-28 更新于 2022-03-25 分类于 硬件
本文字数： 482 阅读时长 ≈ 1 分钟

周末第一次自己动手组装了台电脑，记录一下

需要的零件清单

  • CPU
  • 主板
  • 散热
  • 显卡
  • SSD
  • 内存条
  • 电源
  • 机箱
  • 显示器

选择配置

  • CPU：技嘉i5-11400f
  • 主板：技嘉B560 M
  • 散热：九州风神 玄冰400
  • 显卡：一直用的GTX970，还能再战两年🐶
  • SSD：三星970 EVO Plus
  • 内存条：金士顿 FURY 16GB DDR4 3200频
  • 电源：技嘉P550B 550W电源
  • 机箱：技嘉幽灵C101G
  • 显示器：三星S24D590PL

安装过程

  1. 先把CPU装主板上
  2. 安装CPU散热
  3. 安装SSD和内存条
  4. 把主板安装到机箱里
  5. 把电源放到机箱里走好线
  6. 插线，主板供电，CPU供电，开机跳线
  7. 安装显卡，插显卡供电
  8. 插好显示器和外设的线

装系统

  1. 到大白菜装机官网下载装系统工具
  2. 做u盘启动盘
  3. u盘启动盘插到新主机上启动安装系统
  4. 安装后自动重启即可

遇到问题：装机后重启无法进入系统

解决方案：
修改BIOS的参数：

  1. Secure Boot改成Disabled，禁用安全启动
  2. CSM改成Enable，表示兼容
  3. Boot Mode/UEFI Mode改成Legacy，表示传统启动方式
  4. OS Optimized Defaults改为Disabled，表示禁用默认系统优先配置
