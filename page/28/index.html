flutter学习系列（一）

发表于 2021-06-30 更新于 2021-07-09 分类于 flutter
本文字数： 2.8k 阅读时长 ≈ 3 分钟

介绍

Flutter是谷歌出品的移动应用SDK，它可以制作高质量的具有原生体验的跨Android和iOS平台的应用。也可以基于原有代码进行开发，世界各地广受欢迎，开源且免费。

安装环境

https://flutterchina.club/setup-macos/

  1. https://flutter.dev/docs/development/tools/sdk/releases#macos 获取flutter SDK配置环境变量
  2. https://developer.android.com/studio/index.html 安装Android Studio
  3. 下载Xcode
  4. 运行flutter doctor命令检查环境是否配好

问题记录

运行flutter doctor –android-licenses命令时报错

解决方案：

https://stackoverflow.com/questions/64708446/flutter-doctor-android-licenses-exception-in-thread-main-java-lang-noclass

  1. Open Android Studio
  2. Tools > SDK Manager
  3. From the left choose, Appearance & Behavior > System Settings > Android SDK
  4. Select SDK Tools from the top menu
  5. Check Android SDK Command-line tools and click ‘apply’.

使用vscode创建第一个Flutter app

创建Flutter项目

首先需要安装dart和flutter插件。
然后 快捷键command+shift+p（mac）/ ctrl+shift+p（windows）打开命令面板输入flutter执行Flutter: New Application Project然后输入工程名字即可创建一个Flutter项目

运行Flutter APP

创建完 Flutter APP 的工程后，工程里面有默认的实现，我们可以直接运行 Flutter APP，但在运行 Flutter APP 之前，首先要打开模拟器，或者连接真机。

在vscode的底部工具栏有一个选择设备的选项，默认值为No Devices，点击后就可以选择一个模拟器了。

Flutter的运行有两种模式：

  • Start Debugging

  • Starting Without Debugging

    在debug模式下我们可以在DEBUG CONSOLE里打印日志

目录介绍

  • lib
    这里是Flutter的代码，默认有一个main.dart文件是Flutter的入口文件

  • android
    一个Android工程的代码，是Flutter在Android上的壳子

  • ios
    一个IOS工程的代码，是Flutter在IOS上的壳子

  • test
    Flutter的测试代码目录

  • pubspec.yaml
    Flutter的配置文件，声明了 Flutter APP 的名称、版本、作者等的元数据文件
