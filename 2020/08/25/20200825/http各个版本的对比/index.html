http各个版本的特点和对比

发表于 2020-08-25 分类于 http 阅读次数： Valine：
本文字数： 406 阅读时长 ≈ 1 分钟

http1.0

  • 请求头和响应头
  • Cache缓存机制
  • 每次http请求都要经历三步：建立TCP连接、传输HTTP数据、断开TCP连接

http1.1

  • 持久连接，浏览器中对于同一个域名，默认允许同时建立6个TCP持久连接
  • 存在队头阻塞的问题
  • 增加了Host字段，提供虚拟主机的支持
  • 对动态生成的内容提供了完美支持
  • 客户端Cookie、安全机制

http1.1缺点：

  • 带宽利用率差
  • TCP的慢启动。
  • 同时开启多条TCP连接，那么这些连接会竞争固定的带宽。
  • HTTP/1.1队头阻塞的问题

http2

增加了以下特性：

  • 多路复用
  • 可以设置请求的优先级
  • 服务器推送
  • 头部压缩

多路复用

多路复用是通过在协议栈中添加二进制分帧实现的，它可以将请求分开去传输，这样可以使浏览器收到更高优先级的请求时优先处理

https

http是明文传输，https是加密的http请求，使用SSL/TSL加密，可以对发起HTTP请求的数据进行加密操作和对接收到HTTP的内容进行解密操作。
