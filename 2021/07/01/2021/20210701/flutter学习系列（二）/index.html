Dart入门

因为Flutter是用Dart语言开发的，所以学习Flutter之前有必要先学习一下Dart

Dart简介

Dart是Google开发的语言，可以用于 移动端、PC 端、WEB、服务器的开发，是全栈式编程语言。很多人可能以为Dart是新出的语言，实际上Dart已经有十年历史了。

Dart的优势

  • Dart 中的所有东西都是对象，包括数字、函数等，它们都继承自 Object，并且对象的默认值都是 null（包括数字）。
  • Dart 既可以支持 JIT（动态编译），也可以支持 AOT（静态编译）。
  • Dart 是强类型语言，但是由于 Dart 可以推断类型，所以也可以支持动态类型，例如 var、dynamic。
  • Dart 有强大的异步编程能力。

基本语法

Dart和js/ts的部分区别

  • Dart要写分号
  • Dart里面所有值都是Object，没有联合类型
  • 没有隐式类型转换
  • 没有Array取而代之的是List，Map和Set是两者都有的
  • 实例化类的时候不用new操作符
  • js里在为 yourName 赋值时，若 name 有值，就使用 name 的值，若 name 为空，则使用默认值 Bob，是这么写的： Dart是这样的：
  • Dart里if判断语句中的条件只能是bool类型
