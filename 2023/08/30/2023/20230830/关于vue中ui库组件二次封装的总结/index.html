关于vue中ui库组件二次封装的总结

发表于 2023-08-30 分类于 vue
本文字数： 1.8k 阅读时长 ≈ 2 分钟

背景

在项目开发中需要对组件进行二次封装是很常见的需求，今天来总结一下二次封装组件会遇到的问题

问题

属性和方法绑定

组件封装的时候遇到的第一个问题就是绑定属性和方法了，比如封装一个el-input组件，我们肯定希望将二次封装的组件属性方法都绑定到el-input组件上。

解决方案：

  1. 把组件用到的所有属性和方法一个一个全部定义好，写在封装组件的props里（非常耗时耗力，不易维护）
  2. 通过$attrs实现绑定属性。$attrs 是一个对象，包含了父组件传递给子组件的非 prop 属性。当一个组件接收到一个属性，但并没有在 props 中声明的时候，它会自动将该属性添加到渲染的根元素上。当我们需要将这些属性传递给一个子组件而不是根元素上时就可以使用 $attrs
  1. 通过$listeners实现绑定方法。用法和$attrs一致
