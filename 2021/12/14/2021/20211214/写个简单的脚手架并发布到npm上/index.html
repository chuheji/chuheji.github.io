写个简单的脚手架并发布到npm上

发表于 2021-12-14 分类于 工程化
本文字数： 3k 阅读时长 ≈ 3 分钟

背景

每次新起一个项目都要从原来旧的项目拷贝一份（虽然也不是很麻烦，git clone也可以），所以简单搞个脚手架免去重复工作

可能需要的依赖包

  • commander: 命令行工具
  • download-git-repo: 用来下载远程模板
  • ini: 格式转换
  • inquirer: 交互式命令行工具
  • ora: 显示loading动画
  • chalk: 修改控制台输出内容样式
  • log-symbols: 显示出 √ 或 × 等的图标
  • shelljs: 编写shell脚本

本文用到的库：
chalk ora inquirer shell
